# ⚡ DebloaterPRO

A premium Windows 10/11 debloater with GUI, safety features, and performance tweaks.

## 🔥 Features

* Modern GUI (WinForms)
* Safe & categorized app removal
* Restore point protection
* Dry-run mode (test before apply)
* Profile export/import
* Performance tweaks
* Logging system

## ⚠️ Requirements

* Windows 10/11
* Run as Administrator
* PowerShell 5.1+

## 🚀 Run مباشرة (Direct Run)

```powershell
irm https://raw.githubusercontent.com/YOUR_USERNAME/DebloaterPRO/main/DebloaterPRO.ps1 | iex
```

## 🔐 Safer Method (Recommended)

```powershell
irm https://raw.githubusercontent.com/YOUR_USERNAME/DebloaterPRO/main/DebloaterPRO.ps1 -OutFile DebloaterPRO.ps1
powershell -ExecutionPolicy Bypass -File .\DebloaterPRO.ps1
```
